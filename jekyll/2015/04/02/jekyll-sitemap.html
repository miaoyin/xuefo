  • 查看jekyll版本
    查到原因，可能是jekyll版本太旧. 先看看自己的jekyll版本
➜  blog git:(master) jekyll -v
jekyll 1.5.0
  • 升级jekyll
➜  blog git:(master) sudo gem install jekyll

升级之后jekyll版本为2.5.3，重新启动jekyll，运行正常.
