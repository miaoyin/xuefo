pymode导致vim保存py文件很慢
妙音 posted @ 2015-05-25 11:58 in vim with tags

  • 问题
    vim安装pymode插件后，保存py文件, 出现vim卡住了，vim底部提示"pymode regenerate auto import cache".

  • 解决
    在.vimrc中添加配置关闭自动构建.

" 当前目录文件很多时，会导致第一次保存非常慢
call pymode#default('g:pymode_rope_autoimport', 0)

如果还没有效果，再加上

let g:pymode_rope = 0
