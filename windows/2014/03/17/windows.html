由于帐户限制,您无法登录
妙音 posted @ 2014-03-17 17:47 in windows with tags

远程登陆没有密码的windows系统，出现“由于帐户限制,您无法登录”

解决办法
* 进入本地安全策略:
cmd里运行secpol.msc
* 禁用服务
本地安全策略 --> 安全选项 -->账户：使用空密码的本地账户只允许进行控制台登陆
