shell中怎么注释多行
妙音 posted @ 2015-05-08 16:08 in shell with tags

shell本身没有多行注释，有什么办法达到多行注释效果？

  • 语法
:<<标记

...代码

标记

冒号":"表示什么也不做，即空命令 重定向"<<"是Here Document 表示把文档内容重定向到空命令

  • 示例一
:<<EOF
echo "hello world"
...
EOF
  • 示例二
:<<BLOCK
echo "hello world"
....
BLOCK
