meliae内存分析-基本用法
妙音 posted @ 2014-02-24 12:58 in python with tags

python的内存分析工具meliae，挺好用的.

示例

  1. 记录程序的内存使用信息
 1 import time
 2  
 3 # 此处是可能发生内存泄露的地方
 4 def main():
 5     pass
 6  
 7 if __name__ == '__main__':
 8     # 通过循环执行，将泄露的内存积累放大
 9     while True:
10         main()
11         time.sleep(10)
12         # 记录每次执行完后占用的内存
13         from meliae import scanner
14         scanner.dump_all_objects('mem_dump%s' % time.strftime("%Y-%m-%d %H:%M:%S", time.localtime()))
  1. 查看占用的内存大小
1 from meliae import loader
2 om = loader.load('mem_dump2014-02-2022-34-43')
3 om.summarize()

具体分析引用关系太麻烦，以上是个人喜欢的方法：
用循环将泄露的内存累积放大. 然后记录每次执行完后占用的内存.
查看占用内存大小. 再比较每次的内存大小，即可知道是否发生泄露.
至于内存泄露的地方，则用排除法，重复上述过程，定位泄露地点.

基本用法

获取进程的内存信息

1 import time
2 from meliae import scanner
3 scanner.dump_all_objects('mem_dump%s' % time.strftime("%Y-%m-%d %H:%M:%S", time.localtime()))
  • 查看占用内存
1 from meliae import loader
2 #加载dump文件
3 om = loader.load('mem_dump2014-02-2022-34-43')
4 #分析内存占用情况
5 om.summarize()

字段说明
* Index : 行索引号
* Count : 该类型的对象总数
* %(Count) : 该类型的对象总数 占 所有类型的对象总数 的百分比
* Size : 该类型的对象总字节数
* %(Size) : 该类型的对象总字节数 占 所有类型的对象总字节数 的百分比
* Cum : 累积行索引后的%(Size)
* Max : 该类型的对象中，最大者的字节数
* Kind : 类型
